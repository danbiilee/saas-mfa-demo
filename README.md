# Micro Frontend Architecture - React

## About

Micro Frontend Architecture가 적용된 React 프로젝트입니다.

## Live Demo

https://micro-host.netlify.app/

## Getting Started

```bash
npm i -g lerna
yarn bootstrap && yarn start
```

- Host
  - `./host`: localhost:3000
- Remote
  - `./remotes/app1`: localhost:3001
  - `./remotes/app2`: localhost:3002
